# Model Agency Landing Page

Адаптивный лендинг для модельного агентства на HTML, CSS и JavaScript.

## Live demo

Сайт опубликован и доступен по ссылке:

```text
https://mgmrnd.ru
```

## О проекте

Проект представляет собой многостраничный лендинг для модельного агентства.

На сайте реализованы:

- полноэкранный hero-блок;
- фиксированная навигация;
- мобильное меню;
- секция «О нас»;
- секция «Фотосессии»;
- секция «Требования»;
- контактный блок;
- бегущие строки;
- декоративные изображения и карточки;
- адаптивная вёрстка;
- legal-страницы: политика конфиденциальности и пользовательское соглашение.

## Особенности реализации

- Адаптивная вёрстка под разные размеры экранов.
- Hero-секция на весь экран.
- Фиксированная верхняя навигация.
- Мобильное меню с burger-кнопкой.
- Бесшовная бегущая строка на JavaScript.
- Отдельные legal-страницы.
- Кастомная визуальная стилизация секций.
- Работа с фоновыми изображениями, карточками, рамками и декоративными элементами.

## Примечание

В репозитории размещена очищенная версия исходного кода.

Медиафайлы и локальные шрифты не включены в публичный репозиторий, так как полноценная визуальная версия сайта доступна по live-ссылке.

Это сделано для того, чтобы не публиковать приватные изображения, коммерческие материалы и локальные font-файлы.
